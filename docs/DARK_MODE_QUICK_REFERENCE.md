Ensure component is wrapped by `<ThemeProvider>`\n2. Check `document.body.className` in DevTools\n3. Clear localStorage: `localStorage.clear()`\n4. Hard refresh: `Ctrl+Shift+Delete`\n\n### Issue: Colors not updating\n**Solution:**\n1. Use CSS variables instead of hardcoded colors\n2. Add `transition` property for smooth change\n3. Check CSS specificity issues\n4. Use `body.dark .selector` for overrides\n\n### Issue: Flickering on page load\n**Solution:**\n1. ThemeProvider is called early (it is in App.jsx)\n2. useEffect syncs state to localStorage\n3. Should not have flicker with proper implementation\n\n### Issue: localStorage errors\n**Solution:**\n1. Check if localStorage is available (private browsing)\n2. Context has try-catch for JSON parse\n3. Falls back to false if error\n\n---\n\n## Testing Dark Mode\n\n### Manual Testing\n```javascript\n// In browser console:\n\n// Toggle dark mode\ndocument.body.className =
